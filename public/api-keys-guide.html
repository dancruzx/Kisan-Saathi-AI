How to Get Free AI API Keys

Kisan Saathi runs on your device and connects directly to AI providers. Follow these steps to get your free API keys.

🔥 Recommended: Get a Free Groq API Key

Groq provides ultra-fast inference and offers a generous free tier for developers.

  1. Go to the Groq Console.
  2. Sign in or create a free account using your Google, GitHub, or email.
  3. On the left sidebar, click on API Keys.
  4. Click the Create API Key button.
  5. Give it a name (e.g., "Kisan Saathi") and click Submit.
  6. Copy the generated key (it starts with gsk_).
  7. Paste this key into the Kisan Saathi app.

Google Gemini API Key (via Google AI Studio)

Google offers free API access to their Gemini models through AI Studio.

  1. Go to Google AI Studio.
  2. Sign in with your Google account.
  3. In the left navigation menu, click on Get API key.
  4. Click the Create API key button.
  5. Choose a Google Cloud Project (or let it create a new one) and confirm.
  6. Copy the generated API key.
  7. Paste this key into the Kisan Saathi app.
